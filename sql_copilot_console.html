Note: the live Claude API call in this demo only works when this page is opened inside an Anthropic artifact sandbox (e.g. within a Claude.ai conversation). Opened as a plain downloaded file in a regular browser, the API call will fail -- browsers can't authenticate directly to Anthropic's API without a server acting as an intermediary, and this demo has none. See the companion notebook and backend scaffold in this repo for a version designed to run anywhere.

Runs against a 1,097-row stratified sample (all 447 confirmed Fraud Risk orders + a mix of Return Risk / Normal) of the full 12,000-order dataset, entirely in your browser — no server, no database beyond this sandbox's own API access. SQL generation calls the live Claude API grounded on the real 23-column schema; execution uses a lightweight client-side interpreter supporting SELECT / WHERE / ORDER BY / LIMIT over a single table (no joins, aggregates, or window functions — those are demonstrated in the full notebook pipeline). The sanitizer and schema guardrail are direct ports of the ones validated in that notebook.
